#Interface de test Qt, OpenCv

Complémentaire au projet de simulation de robot sur Unity.

Installation:

Pour pouvoir compiler le projet il faut installer Qt 5.7.1 et OpenCv 3.2.0 minimum.

Utilisation:

Depuis la fenêtres principales, vous pouvez avoir accés à des sous-fenêtres pour pouvoir tester des fonctionnalités
Petite précision, pour pouvoir utiliser les modes test sur robots et simulation, il faut d'abord faire une calibrations des caméras stéréoscopiques et préciser les paramétres pour l'algorithme utilisé (BM ou SGBM). Des exemples des fichiers qui sont nécessaires sont dans le dossier files.
