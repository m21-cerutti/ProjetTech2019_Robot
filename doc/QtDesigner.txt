
Fenetre graphique :

Pour créer une nouvelle fenetre graphique (! si elle existe déjà, copier coller le code existant),
File->new file or project->Qt->Qt designer form class
Choisissez ensuite les options (! choisissez dialog si vous voulez une fenetre).

Ajout widgets QtDesigner :

Glissez et renommer pour y accéder dans le code.
Doit compiler sans erreur une fois pour y accéder.
Mise en page statique peut se faire depuis Qt designer.

Ajout connect et slots QtDesigner :

Pour les menus, les actions sont rajoutés en bas dans "Action editor". Clique droit, aller au slot 
(go to slot).
Pour les widgets, meme principe, Clique droit sur le widget, dans la hiérachie en haut à droite, aller au slot.
Choisir ensuite le signal (trigered souvent) et la fonction va se rajouter automatiquement dans le code. Le connect est ainsi fait aussi.
Peut se faire aussi plus subtilement avec le "signals& slots editor" (sans les slots personnalisées en code, idéal pour des fermetures de fenetre sans code, show,...).
